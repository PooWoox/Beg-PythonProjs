# GeoGame 1.0v
- Esse é um jogo simples estilo quiz. Inicialmente apenas com perguntas das capitais dos estados brasileiros.
- O sistema pesquisará a sigla aleatória de um estado em uma lista com estados e capitais.
- Para computar o acerto ou erro, o sistema comparará o input do usuário com o respectivo valor de capital do estado selecionado.
- Se respondido corretamente o usuário ganhará um ponto, ao errar ele terá outra chance de responder.

Instruções
- Ao executar o arquivo, o usuário receberá uma pergunta que incluirá um estado aleatório do Brasil e deverá responder com sua respectiva capital.
- A resposta deve ser dada no padrão de nome próprio, iniciando o nome da capital em letra maiúscula. Ex: "São Paulo" será aceito e "são paulo" não.
